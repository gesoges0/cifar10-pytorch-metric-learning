# cifar10-pytorch-metric-learning

metric learning for cifar10 [3channels images]

Metric Learning を3チャネル画像で行うためのプログラム

- Network構造

  - Triplet Newtowrk |
- 損失関数
  - Contrastive Loss
  - Online Contrastive Loss
  - Triplet Loss

  - Ranked List Loss
  - N-pair MC Loss
